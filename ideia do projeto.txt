APi de estudos, uma api de bibliotecas que controla usuarios, os deixa criar contas, com email e senha,faz o login deles, 
os deixa escolherem livros para empréstimos por X dias definidos por eles,
permite tambem o CRUD de novos livros por usuarios administradores logados, apenas usuarios administradores podem promover 
outros a administrador e o primeiro usuario sempre será administrador

web framework -> Gin

banco -> GORM - postgress 
comando docker: docker run --name postgres-library -e POSTGRES_PASSWORD=postgres -e POSTGRES_USER=postgres -e POSTGRES_DB=library_api -p 5432:5432 -d postgres

autenticação -> gin-jwt

validação de dados -> Gin

configuração -> godotenv
